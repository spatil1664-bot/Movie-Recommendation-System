# 🎬 Movie Recommendation System

## 📌 Overview
This is a Movie Recommendation System built using Python and NLP techniques. It recommends similar movies based on content using TF-IDF Vectorization and Cosine Similarity.

## ✨ Features
- Recommend similar movies
- Content-based filtering
- User-friendly Streamlit interface
- Fast movie search

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity
- Streamlit

## 🚀 How to Run
1. Install the required libraries.
2. Run:
   ```
   streamlit run 03_app.py
   ```
3. Select a movie and get recommendations.
